## Overview

This project is a menu-driven Expense Tracker program written in C that allows users to record and manage daily expenses.
The program stores expense details in a file and provides options to view and analyze expenses based on category and date.

The project is also compiled for browser execution using WebAssembly (Emscripten) and hosted via GitHub Pages.

## Features
Add expense with date, category, amount, and note
View all stored expenses
Calculate monthly expense total
Calculate category-wise expense total
Search expense by date
Search expense by category
File-based data storage using expenses.txt
Technologies Used
C Programming Language
File Handling in C
WebAssembly (Emscripten)
HTML (for browser interface)
GitHub Pages (deployment)

## File Storage

All expense records are saved in:
expenses.txt

This file is automatically created when adding the first expense.

## Limitations

Browser version has limited support for interactive terminal input due to WebAssembly environment constraints.
Full functionality can be demonstrated using the terminal version.
